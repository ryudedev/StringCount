# StringCount

## インストール方法
1. cloneする
```bash
git clone https://github.com/ryudedev/StringCount.git
```
2. サーバーを起動させる
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

3. ブラウザを確認する。
[https://localhost:3000](https://localhost:3000)
